# AquaTwin

AquaTwin is a prototype water digital twin dashboard built with:

- Google Earth Engine Python API
- Streamlit
- geemap

## Features
- Lake Victoria monitoring
- Water extent using NDWI
- Chlorophyll/algae proxy using NDCI-style index
- Catchment vegetation condition using NDVI
- Rainfall monitoring using CHIRPS
- Prototype environmental risk score
